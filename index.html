How to implement

Implementing the verification service is easy!
  • Add this javascript to your page: <script src="http://clearsign.me/clearsign.js"/></script>
  • Wrap your GPG signature blocks on your pages with <div class=".gpg-signed-text"></div>

Alternatively you can check this project out from github and run the verification service yourself.

https://github.com/arthur24b6/clearsign
